# 同等学力申硕考试备考资料

> 同等学力人员申请硕士学位全国统一考试备考资料库，涵盖**英语**和**计算机科学与技术**两个科目。

## 项目简介

本项目整理了同等学力申硕考试的完整备考资料，包括系统化的复习计划、知识点总结、写作模板与专项训练等内容，旨在帮助考生高效备考，在短时间内掌握考试要点。

## 英语备考

### 考试概况

- 总分 100 分，60 分及格
- 考试时间 150 分钟（卷一 75 分钟 + 卷二 75 分钟）
- 题型：口语交际、词汇、阅读理解、完形填空、短文完成、英译汉、写作

### 备考计划特色

| 计划 | 内容 | 时长 |
|------|------|------|
| 一个月备考计划 | 4周分阶段备考（基础→题型→综合→冲刺） | 30天 |
| 30天每日任务总表 | 词汇+词组+写作+备考四合一整合版 | 30天 |
| 30天单词分日计划 | 按主题分类，约6000词 | 30天 |
| 30天词组分日计划 | 按动词分组，约700词组 | 30天 |
| 写作专项突破计划 | 句子→段落→模板→范文，冲击10+分 | 贯穿全程 |

### 四周复习安排

1. **第一周**：基础夯实期 — 词汇+语法为主
2. **第二周**：题型突破期 — 各题型逐一攻克
3. **第三周**：综合提升期 — 翻译+写作+真题训练
4. **第四周**：冲刺模拟期 — 全真模拟+查漏补缺

## 计算机备考

### 考试概况

- 总分 100 分
- 科目组成：数学基础（40分，必答）+ 软件工程（30分）+ 人工智能（30分）

### 四周复习安排

1. **第一周**：攻克三大必考大题（谓词符号化、DFD数据流图、归结推理）
2. **第二周**：补齐中频知识点 + 强化练习
3. **第三周**：真题套卷演练 + 薄弱点突破
4. **第四周**：全真模拟 + 冲刺记忆

### 核心必考点

| 科目 | 必考点 | 分值 |
|------|--------|------|
| 数学基础 | 谓词符号化、排列组合、等价/偏序关系、母函数与递推 | 40分 |
| 软件工程 | DFD数据流图（10分必考）、内聚耦合、面向对象、过程模型 | 30分 |
| 人工智能 | 归结推理证明（8分必考）、A*搜索、神经网络、不确定推理 | 30分 |

## 使用建议

- 每天学习 3.5-5.5 小时，可根据个人情况调整
- 严格按照计划表的顺序推进，先攻克必考点再补充中频点
- 真题是最重要的练习资源，至少做 4-5 套近年真题
- 建立错题本，定期回顾反复出错的题型
