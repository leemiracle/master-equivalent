#### Day 1-2：谓词符号化 + 排列组合基础

**Day 1 上午（2h）— 谓词符号化**

学习内容：
1. 全称量词 ∀ 和存在量词 ∃ 的基本用法
2. 标准句型转换模板（必背）：
   - "所有 S 都是 P" → ∀x(S(x)→P(x))
   - "有些 S 是 P" → ∃x(S(x)∧P(x))
   - "不是所有 S 都是 P" → ¬∀x(S(x)→P(x)) ≡ ∃x(S(x)∧¬P(x))
   - "没有 S 是 P" → ¬∃x(S(x)∧P(x)) ≡ ∀x(S(x)→¬P(x))
   - "有且仅有一个 S" → (∃x)(S(x)∧∀y(S(y)→y=x)) 或 (∃!x)S(x)
3. 四步解题法：定义谓词 → 确定论域 → 用∀写 → 用∃写

练习：
- [ ] 2025真题：没有两个人指纹完全相同
- [ ] 2024真题：成绩达标的人才有资格参加奥运会
- [ ] 2022真题：任意一个自然数，有且只有一个后继
- [ ] 2020真题：确诊者并不都有症状
- [ ] 2018真题：天下没有长相完全一样的两个人
- [ ] 2012真题：有且仅有一个火星

**Day 1 下午（2h）— 排列组合基础**

学习内容：
1. 基本公式：P(n,r) = n!/(n-r)!，C(n,r) = n!/(r!(n-r)!)
2. 全排列、不全相异元素排列 n!/(n₁!n₂!…)
3. 加法法则、乘法法则
4. 二项式定理 (a+b)ⁿ = ΣC(n,k)aⁿ⁻ᵏbᵏ

练习：
- [ ] 2012真题：3个a、1个b、2个c的排列数（60）
- [ ] 2024真题：方程x₁+…+x₆=7的非负整数解（792）和正整数解（6）
- [ ] 2025真题：从10名志愿者选7人，甲乙至少一人入选（112）
- [ ] 2020真题：C(n,1)+…+C(n,n)的值（2ⁿ-1）
- [ ] 2024真题：(x₁+x₂+x₃)⁵中x₁²x₂²x₃系数（30）

**Day 2 上午（2h）— 排列组合进阶题型**

学习内容（6大题型，每种记公式+方法）：
1. **男女相间排列**：2×(m!)²；限定女生末位：(m!)²
2. **插排法（不相邻）**：先排其他人，再插入间隙 P(6,3)
3. **圆排列**：(n-1)!；绑定相邻：×2ᵏ（k对夫妻内部交换）
4. **分组分配**：先分组再排列，无序组÷组数阶乘
5. **隔板法**：非负整数解 C(n+k-1,k-1)，正整数解 C(n-1,k-1)
6. **排除法**："至少一人"= 总数 - 都不选

练习：
- [ ] 2024真题：m男m女相间且女生末位（(m!)²）
- [ ] 2021真题：5男3女，女生不相邻（14400）
- [ ] 2013真题：4对夫妻圆排列（5040）/ 相邻（96）
- [ ] 2023真题：5师3生圆桌，学生不相邻（1440）
- [ ] 2025真题：8人选6人分3组训练（420）
- [ ] 2022真题：11名同学分4组（11550）

**Day 2 下午（2h）— 归结推理入门**

学习内容：
1. 命题逻辑归结：前提+结论否定 → CNF → 归结出空子句□
2. 谓词逻辑归结：Skolem化 → 消去量词 → CNF → 合一+归结
3. 合一（Unification）：找替换θ使两个文字相同
4. 归结策略：单元归结（不完备）、线性归结、支持集策略

练习：
- [ ] 简单例题：(P→Q)∧(Q→R) ⊢ P→R

#### Day 3-4：等价关系/偏序关系 + DFD 数据流图

**Day 3 上午（2h）— 等价关系与偏序关系**

学习内容：
1. 等价关系三条件：自反+对称+传递
2. 等价关系与划分一一对应，有序对数 = Σ|Sᵢ|²
3. 偏序关系：自反+反对称+传递
4. 链（两两可比）、反链（两两不可比）
5. 整除关系的分析：逐元素统计+最长链/反链
6. 既是等价又是偏序 → 只有恒等关系

练习：
- [ ] 2025真题：A={1,…,10}整除关系，27个有序对，最长反链5，最长链4
- [ ] 2024真题：A={2,…,12}整除关系的偏序分析

- [ ] 2023真题：证明自反+"若<a,b>,<a,c>∈R则<b,c>∈R"→等价关系

**Day 3 下午（2h）— 二元关系运算 + 图论**

学习内容：
1. 复合运算R₂∘R₁（先R₁后R₂）、逆关系R⁻¹
2. 关系性质：自反/反自反/对称/反对称/传递（矩阵特征）
3. 传递闭包计算：逐步添加
4. 邻接矩阵：n×n方阵，Aᵏ的(i,j)元素=长度k的通路数
5. 欧拉公式：V-E+F=2（连通平面图）
6. 生成树：破圈法/避圈法

练习：
- [ ] 2024真题：R={<1,2>,<2,3>,<3,4>}的传递闭包（6个有序对）
- [ ] 2019真题：R₁,R₂的复合运算
- [ ] 2018真题：非自反+传递→反对称（证明题）
- [ ] 2012真题：生成树棵数（破圈法）

**Day 4 全天（4h）— DFD 数据流图（10分必考大题）**

学习内容（上午）：
1. DFD 四种基本元素：外部实体（矩形）、加工（圆角矩形）、数据流（箭头）、数据存储（双横线）
2. 分层结构：顶层图→0层图→1层图
3. **七条核心规则**（必记）：
   - 数据守恒：输入→加工→输出，不凭空产生/消失
   - 父子图平衡：父图加工边界=子图整体边界
   - 加工必须编号（0→1,2,3→1.1,1.2）
   - 数据流必须命名（名词/名词短语）
   - 数据存储只能被加工读写
   - 分解3-7个子加工
   - 避免黑洞/奇迹/灰洞
4. 常见错误类型：黑洞、奇迹、灰洞、外部实体直连存储

练习（下午，每题限时15分钟）：
- [ ] 2025真题建模题：销售管理系统（顶层→0层→1层DFD）
